# Portfolio Risk Analytics Dashboard

Live Application:  
https://stockdashboardd.streamlit.app/

---

## Overview

Portfolio Risk Analytics Dashboard is an institutional-style financial analytics platform built using Python, Streamlit, Plotly, and Yahoo Finance data.

The application provides interactive portfolio analysis, benchmark comparison, advanced risk metrics, simulation models, and downloadable PDF reporting. It is designed to simulate a real-world Asset Management analytics environment.

---

## Key Features

### Performance Analytics
- Portfolio vs Benchmark cumulative return comparison
- Individual stock performance tracking
- Dynamic portfolio weight adjustment
- Real-time KPI updates

### Risk Metrics
- Annualized Return
- Volatility
- Sharpe Ratio
- Beta
- Tracking Error
- Maximum Drawdown
- Rolling 30-day volatility
- Rolling 30-day beta

### Advanced Insights
- Correlation matrix heatmap
- Return distribution analysis
- Portfolio vs Benchmark scatter analysis

### Simulation Engine
- Live rebalancing simulation
- Monte Carlo simulation for future portfolio projection
- Configurable rebalance frequency

### Portfolio Allocation
- Dynamic allocation pie visualization
- Weight normalization logic

### Reporting
- One-click downloadable PDF risk report
